# Weather app
## Frameworks, components, libraries, architecture and other technologies
- Kotlin
- Started with MVP
- Converted to MVVM
- Retrofit 2
- Picasso/Coil/Glide
- Build variants (Flavours/Types)
- Firebase
- Room
- Service/BroadcastReceiver/Content Provider
- RxJava
## Data source
[https://api.weather.yandex.ru/](https://yandex.ru/dev/weather/)
